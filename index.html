3

Setelah proses selesai, klik tombol download untuk mengunduh media

API Documentation

Gunakan API ini langsung melalui endpoint berikut:

GET /api/pinterest?url={pinterest_url}

Contoh penggunaan dengan JavaScript:

fetch('/api/pinterest?url=https://pin.it/1sFZsJRDZ') .then(response => response.json()) .then(data => console.log(data));

Response JSON akan berisi:

{ "success": true, "data": { "mediaUrls": { "videos": [ "https://v1.pinimg.com/videos/.../video.mp4" ], "images": [ "https://i.pinimg.com/.../image.jpg" ] }, "primaryUrl": "https://v1.pinimg.com/videos/.../video.mp4", "type": "video", "sourceUrl": "https://pin.it/1sFZsJRDZ" } }
