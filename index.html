API Documentation

Anda juga dapat mengakses API ini langsung melalui endpoint berikut:

GET /api/threads?url={threads_url}

Contoh penggunaan dengan curl:

curl -X 'GET' \
  'https://your-vercel-app.vercel.app/api/threads?url=https%3A%2F%2Fwww.threads.net%2F%40username%2Fpost%2FpostId' \
  -H 'accept: application/json'

Response JSON akan berisi:

{
  "success": true,
  "data": {
    "post_detail": {
      "code": "POST_ID",
      "caption_text": "Caption postingan...",

      "media_list": [
        {
          "media_type": 2,
          "url": "https://video-url.mp4",

        }
      ]
    },
    "user": {
      "username": "username",
      "full_name": "User Full Name",
      "avatar": "https://avatar-url.jpg"
    }
  }
}
